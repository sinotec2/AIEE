## What's New

- 2025-12-23 聖誕節前夕來回顧這一兩年的變化，算是黑夜中對星光的渴望吧。LLM更新的速度好像巨浪海嘯般襲來，不單衝擊著一般消費者、公司同仁的使用習慣經驗，也押著公司內部程式系統要在短時間內有效的升級。我總結成三個大的研發題目成為來年的努力的方向：人資系統的建立與營運、備標協作資訊叢集與應用之串接、第三個就是MIS系統的智慧與自動化。近期新增包括：[文檔內嵌](./NLP/Embedding.md)、[學習型機器人](NLP/Embedding/learningRobot.md)、[2025主管AI工作坊](./Reviews/SES_wrokshop1.md)、[ollama監控](./utilities/OpsChatUI/ollama_monitoring.md)。
- 2023-09-17 一個秋老虎的日子來開張這個主題的網誌算是蠻合適的溫度，可以呈現出這個領域毒辣兇狠的成長氣氛。成立的宗旨在於把散落在各處沒有歸屬的程式筆記好好整理一下，怕日子久，連結漸漸淡忘就散失了。
