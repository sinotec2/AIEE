# AntiGravity

{: .no_toc }

{: .fs-6 .fw-300 }

### 關鍵要點

- **Google Antigravity** 被描述為以開發者信任為導向的代理式開發平台，並針對「代理優先」的時代設計。
- 其核心特徵是 **Agent-First / Agentic Coding**：讓 AI 代理不只輔助建議，而是可自主規劃並執行部分開發任務。
- 多篇介紹指出它搭載 **Gemini 3**（或 Gemini 3 Pro 等版本）作為能力底座，支撐程式生成與任務完成。
- 報導提到 **瀏覽器自動化 / Chrome 操控**：代理可模擬使用者行為並操作網頁流程。
- 與其他 AI IDE（如 Cursor）的差異之一在於：Antigravity 強調「指揮與完成工作」的代理模式，而非單純的輔助編輯。
- 文件多處聚焦 **費用與配額機制**：包含 Pro 額度大幅受限、以及配額用罄後可購買點數或彈性加購的做法。
- 有新聞指出 **產品或收費調整引發社群炎上**，例如 Pro 額度突然下修、引發開發者不滿與替代工具轉向。
- 也有內容強調 **整合與擴展**：例如與 **Firebase** 結合，讓從提示生成原型進一步具備儲存、登入與多人互動等能力。
- 官方與教學類資源則提供 **環境建置、Agent 使用與安全考量**，並提到可用 Google Cloud 抵免額來協助在雲端專案落地。
