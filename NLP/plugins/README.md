## 這個主題的宗旨

1. 讓使用者在瀏覽網頁時有更方便的AI小工具、串接網頁瀏覽與筆記之作業流程，不必再另外切換APP。👩‍💼🔗🌐
2. 提供資源：在付費會員之外提供API金鑰、地端Ollama🦙連接方式等資源。💰💰💰
3. 提高語言模型的使用率，不論雲端([Sider](./Sider.md)📖)或者地端([Page Assist](./PageAssist.md)👩‍💼)。👍👍👍
