# LibreChat

{: .no_toc }


![](pngs/Pasted%20image%2020251030120129.png)

==LibreChat== 是一個開源應用程式，旨在提供一個用於 AI 對話的統一平台。它支援多種 AI 供應商，例如 OpenAI 和 Azure OpenAI，並提供**文本轉語音**和**語音轉文本**等功能，讓用戶可以免手持進行對話。它還支援音訊的自動傳送和播放。用戶可以透過**本地安裝**或**遠端託管**來使用 ==LibreChat==。
